**최종 개정일: 2026년 5월 9일**
**시행일: 2026년 5월 9일**

## 1. 수집하는 개인정보 항목

회사는 회원가입, 서비스 제공, 고객 상담, 안전한 서비스 운영을 위해 아래 항목을 수집합니다.

### 1-1. 회원가입 및 계정 관리 (필수)
| 항목 | 수집 시점 | 수집 방법 |
|---|---|---|
| 이메일 주소 | 회원가입 시 | 사용자가 직접 입력 |
| 비밀번호 (해시값) | 회원가입 시 | 사용자가 직접 입력 / 자동 해시 저장 |
| 이름 | 회원가입 시 | 사용자가 직접 입력 |
| 휴대폰 번호 | 회원가입 시 | 사용자가 직접 입력 |

### 1-2. 서비스 이용 과정에서 사용자가 직접 입력하는 정보
| 항목 | 수집 시점 |
|---|---|
| 고객 이름·연락처·주소 | 사장님이 고객 추가 시 |
| 작업 일정·메모·작업 사진 | 작업 등록·완료 시 |
| 매출 금액·결제수단(카드/현금/계좌이체 등 텍스트) | 작업 완료 시 |
| 직원 이름·연락처·인센티브 비율 | 직원 추가 시 |

### 1-3. 자동으로 수집되는 정보
| 항목 | 수집 방법 | 비고 |
|---|---|---|
| 디바이스 정보 (모델명, OS, 앱 버전) | 자동 | Sentry/Expo |
| IP 주소, 접속 일시 | 자동 | Supabase 로그 |
| 크래시 로그·에러 진단 | 자동 (앱 충돌 시) | Sentry |
| 푸시 알림 토큰 | 알림 권한 허용 시 | Expo Push |

### 1-4. 수집하지 않는 정보 (명시적 비수집)
- 광고 식별자(ADID/IDFA)
- GPS 위치 정보 (주소는 사장님이 텍스트로 직접 입력)
- 마이크 음성, 통화 기록, 문자 메시지
- 결제 카드 번호·CVC (회사는 카드 정보를 일체 저장하지 않으며, 결제수단은 "카드/현금/계좌이체" 등 텍스트 분류만 기록함)

### 1-5. 연락처 접근 권한 (선택 권한)
앱은 사장님이 신규 고객을 추가할 때 휴대폰 연락처에서 이름·전화번호를 빠르게 불러올 수 있도록 `READ_CONTACTS` 권한을 요청합니다.
- 권한 거부 시에도 모든 핵심 기능 이용 가능
- 가져온 연락처는 사장님이 명시적으로 "고객으로 추가" 버튼을 눌렀을 때만 회사 서버로 전송
- 연락처 전체를 일괄 업로드하지 않음

## 2. 개인정보의 처리 목적

회사는 수집한 개인정보를 다음의 목적을 위해 처리합니다.

1. **회원 식별 및 인증** — 이메일 기반 로그인, 본인 확인
2. **서비스 제공** — 예약·고객·매출·인센티브·일정 관리 기능
3. **고객 지원** — 문의 응대, 공지사항 전달
4. **앱 안정성 확보** — 충돌 진단, 성능 모니터링
5. **푸시 알림** — 작업 30분 전 리마인드 등 사용자가 허용한 알림
6. **법령상 의무 이행** — 전자상거래법·세법 등에 따른 보관 의무

회사는 위 목적 이외의 용도로는 개인정보를 처리하지 않으며, 목적이 변경되는 경우 별도의 동의를 받습니다.

## 3. 개인정보의 보유 및 이용기간

| 구분 | 보유기간 | 근거 |
|---|---|---|
| 회원 정보 | 회원 탈퇴 시까지 | 정보주체 동의 |
| 작업·매출·고객 데이터 | 회원 탈퇴 후 즉시 파기 (단, SoftDelete 90일 유예) | 정보주체 동의 |
| 결제·세금계산서 관련 기록 | 5년 | 전자상거래법 |
| 부정 이용 기록 | 1년 | 정보주체 동의 |
| Sentry 크래시 로그 | 90일 후 자동 삭제 | Sentry 정책 |

회원 탈퇴 시 모든 데이터는 90일 이내 완전 삭제됩니다(SoftDelete 정책에 따라 90일간 복구 가능 상태로 보관 후 영구 파기).

## 4. 개인정보의 제3자 제공

회사는 정보주체의 개인정보를 제1조의 수집 목적 범위 내에서만 처리하며, 정보주체의 동의·법률의 특별한 규정 등 「개인정보 보호법」 제17조 및 제18조에 해당하는 경우에만 개인정보를 제3자에게 제공합니다.

현재 회사는 광고·마케팅 목적으로 개인정보를 제3자에게 제공하지 않습니다.

## 5. 개인정보의 처리위탁 (해외 이전 포함)

회사는 원활한 서비스 제공을 위해 다음과 같이 개인정보 처리업무를 위탁하고 있습니다.

| 수탁자 | 위탁업무 | 이전 국가 | 이전 항목 | 보유 기간 |
|---|---|---|---|---|
| Supabase, Inc. | 데이터베이스, 인증, 파일 저장소 운영 | 미국 (AWS us-east-1) | 1조의 모든 항목 | 회원 탈퇴 시까지 |
| Functional Software, Inc. (Sentry) | 앱 충돌 로그 수집·분석 | 미국 | 디바이스 정보, 크래시 로그, 사용자 ID | 90일 |
| Expo, Inc. | 푸시 알림 토큰 발급, OTA 업데이트 | 미국 | 디바이스 푸시 토큰 | 회원 탈퇴 시까지 |

이전 일시 및 방법: 서비스 이용 중 네트워크를 통한 실시간 전송 (HTTPS/TLS 암호화)
이전 거부 권리: 정보주체는 위 이전을 거부할 수 있으며, 거부 시 서비스 이용이 제한될 수 있습니다.

## 6. 정보주체와 법정대리인의 권리·의무 및 행사방법

정보주체는 회사에 대해 언제든지 다음의 권리를 행사할 수 있습니다.

1. 개인정보 열람 요구
2. 오류 등이 있을 경우 정정 요구
3. 삭제 요구
4. 처리정지 요구
5. 회원 탈퇴

권리 행사는 앱 내 "설정 → 계정 → 회원 탈퇴" 메뉴 또는 아래 개인정보 보호책임자 연락처로 서면·전자우편으로 요청하실 수 있으며, 회사는 지체 없이 조치하겠습니다.

---

## 7. 개인정보의 파기

회사는 개인정보 보유기간의 경과, 처리목적 달성 등 개인정보가 불필요하게 되었을 때에는 지체 없이 해당 개인정보를 파기합니다.

- **파기 절차**: 회원 탈퇴 요청 → SoftDelete 처리(90일) → 영구 파기
- **파기 방법**:
  - 전자적 파일: 복구·재생할 수 없도록 영구 삭제
  - 출력물: 분쇄 또는 소각

---

## 8. 개인정보의 안전성 확보조치

회사는 개인정보의 안전성 확보를 위해 다음과 같은 조치를 취하고 있습니다.

1. **관리적 조치**: 내부관리계획 수립·시행, 정기적 직원 교육
2. **기술적 조치**:
   - 비밀번호 단방향 해시 저장 (bcrypt)
   - 데이터베이스 RLS(Row Level Security) 적용으로 사장님은 본인 데이터만 조회·수정 가능
   - 모든 통신 HTTPS/TLS 1.2 이상 암호화
   - 접근 권한 관리, 침입차단 시스템 운영
3. **물리적 조치**: Supabase/AWS 데이터센터 물리적 보안 위탁
